# Bootcamp MEAN
## MEAN
### Express v3
#### Socket.io
##### Juego del botón épico

~~~
Para esta asignación, crea la aplicación descrita en el prototipo de abajo utilizando Express y sockets. Tu aplicación debería solo servir una vista, index.ejs. Cada vez que se presiona el botón épico, el contador debe actualizar cada usuario que está conectado a través de sockets. Esto debe pasar en tiempo real. Si un usuario hace clic en el botón de reinicio, el contador debe restablecerse a 0 por cada usuario también.
~~~

~~~
Piensa en los eventos que el cliente necesitará detectar y los eventos que el servidor debe detectar. Asigna los emisores y detectores que necesitarás antes de comenzar a codificar. Usa una variable en el servidor para realizar un seguimiento a cuántas veces se ha hecho clic en el botón.
~~~
